# クライアント向けFAQ（よくある質問）

LINE入退室通知システムに関して、クライアント様からよくいただく質問とその回答をまとめています。

---

## 📱 LINE公式アカウントについて

### Q1. LINE公式アカウントは有料ですか？
**A.** 無料プランから始められます。

- **フリープラン（無料）**: 月200通まで無料
  - 小規模教室（10〜20名程度）なら十分
  - 例: 10名の生徒 × 入退室2回/日 × 20営業日 = 400通/月
- **ライトプラン（5,000円/月）**: 月5,000通まで
- **スタンダードプラン（15,000円/月）**: 月30,000通まで

**推奨:** まずは無料プランで開始し、必要に応じてアップグレード

### Q2. LINE公式アカウントの作成は難しいですか？
**A.** いいえ、簡単です。

- 所要時間: 約10〜15分
- 必要なもの: LINEアカウント、メールアドレス
- 開発者側で代行作成も可能です

### Q3. 既存のLINE公式アカウントを使えますか？
**A.** はい、使用できます。

既存のアカウントがある場合は、そちらを利用することで保護者様が新たに友だち追加する手間が省けます。

---

## 👨‍👩‍👧‍👦 保護者の登録について

### Q4. 保護者はどうやって登録するのですか？
**A.** 3ステップで完了します。

1. **LINE公式アカウントを友だち追加**
   - QRコードをスキャン（約10秒）
2. **管理者が紐付け設定**
   - 管理画面またはAPI経由で親御さんとお子様を紐付け（約1分）
3. **完了**
   - お子様が入退室するとLINEに通知が届きます

### Q5. 両親とも通知を受け取れますか？
**A.** はい、可能です。

- お母様・お父様それぞれがLINE公式アカウントを友だち追加
- 管理者が両方のアカウントをお子様と紐付け
- 両方のLINEに通知が届きます

### Q6. 兄弟姉妹がいる場合はどうなりますか？
**A.** 1つのLINEアカウントで複数のお子様の通知を受け取れます。

例：
```
太郎さんが入室しました。
時刻: 2026/01/06 15:30

花子さんが入室しました。
時刻: 2026/01/06 15:32
```

### Q7. 祖父母にも通知を送れますか？
**A.** はい、可能です。

お子様1人に対して複数の保護者を紐付けできます。
- 母親
- 父親
- 祖母
- 祖父
など、何人でも紐付け可能です。

## 🎴 NFCカードについて

### Q8. どんなカードが使えますか？
**A.** 以下のカードが使用できます。

✅ **使用可能:**
- Suica / PASMO（交通系ICカード）
- FeliCa対応のカード
- 専用NFCカード（Mifare系）

❌ **使用不可:**
- 磁気カード
- バーコードカード

### Q9. 既に持っているSuicaは使えますか？
**A.** はい、使用できます。

生徒が既に持っているSuica/PASMOをそのまま使用できます。新たにカードを購入する必要はありません。

### Q10. カードを忘れた場合はどうなりますか？
**A.** 管理画面から手動で入退室を記録できます。

また、予備のカードを用意しておくこともできます。

### Q11. カードを紛失した場合はどうすればいいですか？
**A.** 管理画面で無効化できます。

1. 紛失したカードを無効化
2. 新しいカードを登録
3. 即座に切り替え完了

## 📢 通知について

### Q12. 通知のタイミングは変更できますか？
**A.** はい、設定可能です。

- 入室時のみ
- 退室時のみ
- 入退室両方
- 自動退室時のみ

など、ニーズに合わせてカスタマイズできます。

### Q13. 通知メッセージの内容は変更できますか？
**A.** はい、自由にカスタマイズできます。

**デフォルト:**
```
太郎さんが入室しました。
時刻: 2026/01/06 15:30
```

**カスタマイズ例:**
```
【〇〇塾】太郎さんが入室しました。
時刻: 2026/01/06 15:30
場所: 本校
獲得ポイント: +10pt（合計: 120pt）
```

### Q14. 通知が届かない場合はどうすればいいですか？
**A.** 以下を確認してください。

1. LINE公式アカウントがブロックされていないか
2. 保護者とお子様が正しく紐付けられているか
3. LINEアカウントが「有効」になっているか

管理画面から通知履歴を確認できます。

## 🔒 セキュリティ・プライバシー

### Q15. 他の保護者に我が子の入退室情報が見られますか？
**A.** いいえ、見られません。

各保護者は**自分のお子様の情報のみ**閲覧できます。他の生徒の情報は一切見られません。

### Q16. 個人情報は安全に管理されますか？
**A.** はい、安全に管理されます。

- データは暗号化されたデータベースに保存
- アクセス権限は厳密に管理
- Supabase（セキュリティ認証取得済み）を使用
- 定期的なバックアップ

### Q17. 退会した生徒のデータはどうなりますか？
**A.** 以下の対応が可能です。

- **アーカイブ**: データを残すが非表示にする（推奨）
- **完全削除**: データを完全に削除する

## 💰 費用について

### Q18. システムの利用料はいくらですか？
**A.** 以下の費用がかかります。

| 項目 | 費用 | 備考 |
|------|------|------|
| **開発費** | 要相談 | 初期開発費用 |
| **LINE公式アカウント** | 0円〜 | 無料プランから |
| **サーバー（Vercel）** | 0円〜 | 無料枠で十分 |
| **データベース（Supabase）** | 0円〜 | 無料枠で十分 |
| **ドメイン（任意）** | 年間1,000円〜 | 独自ドメインを使用する場合 |

**小規模教室（20名以下）の場合: ほぼ無料で運用可能**

### Q19. 生徒数が増えた場合、追加費用はかかりますか？
**A.** 基本的にかかりません。

ただし、以下の場合は有料プランへの移行が必要です：
- LINE通知数が月200通を超える場合（ライトプラン: 5,000円/月）
- データベースの容量が500MBを超える場合（Supabaseの有料プラン）

## 🎯 運用について

### Q20. 導入までどのくらい時間がかかりますか？
**A.** 最短1週間で運用開始できます。

**スケジュール例:**
- **Day 1**: 情報収集・LINE設定
- **Day 2-3**: データ投入・カード登録
- **Day 4-6**: テスト運用
- **Day 7**: 本番運用開始

### Q21. 管理画面の使い方は難しいですか？
**A.** いいえ、シンプルで分かりやすいUIです。

基本的な操作：
- 生徒の追加・編集
- 保護者の紐付け
- 入退室履歴の確認
- ポイント管理

**マニュアルとトレーニングを提供します。**

### Q22. 営業時間外に入退室した場合はどうなりますか？
**A.** 以下の設定が可能です。

- **通常通り記録**: 営業時間外でも通知を送る
- **通知なし**: 営業時間外は記録のみで通知しない
- **エラー表示**: 営業時間外の入退室を禁止する

### Q23. 自動退室とは何ですか？
**A.** 指定時間後に自動で退室処理する機能です。

例：
- 入室後4時間経過で自動退室
- 営業終了時刻（22時）に一括退室

退室忘れを防ぐことができます。

## 🔧 カスタマイズ・拡張

### Q24. ポイントシステムは使えますか？
**A.** はい、使用できます。

既に実装済みの機能：
- 入室時のポイント付与
- 目標達成時のボーナスポイント
- ポイントランキング
- ポイント履歴の確認

### Q25. 将来的に機能を追加できますか？
**A.** はい、可能です。

システムは拡張性を考慮して設計されています。

**追加可能な機能例:**
- 授業予約システム
- 宿題提出管理
- 成績管理
- 請求書発行
- 出席率レポート

### Q26. スマホアプリ版は作れますか？
**A.** はい、作成可能です。

現在はWebアプリですが、以下の形式で展開できます：
- iOS/Androidネイティブアプリ
- PWA（Webアプリをアプリっぽく表示）
- LINEミニアプリ

## 🆘 トラブル対応

### Q27. システムが動かなくなった場合はどうすればいいですか？
**A.** サポート体制を整えています。

- メール・電話でのサポート
- リモートでの緊急対応
- 定期的なメンテナンス

### Q28. データのバックアップはされていますか？
**A.** はい、自動的にバックアップされます。

- Supabaseが自動でバックアップ
- 過去7日間のデータを復元可能
- 手動バックアップも可能

### Q29. 停電やインターネット障害が起きた場合は？
**A.** 以下の対応があります。

- **復旧後**: 自動的に再接続
- **記録**: オフライン時の入退室は手動で記録可能
- **通知**: 後からまとめて送信することも可能

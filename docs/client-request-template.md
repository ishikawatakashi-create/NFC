## 📅 後日でも構わない情報

以下は、開発を進めながら後日お伺いしても大丈夫です：

- 営業時間と自動退室の設定
- ポイントシステムの詳細
- プライバシー設定（入退室情報の公開範囲）
- 専用ドメインの有無
